

Programming Assignment :: Read from Log
-------------------------------------------------

This standalone applicaiton takes the log file, extracts data and displays as expected.
It's a Eclipse maven project, IDE can be used to import the project and execute

To run the applicaiton 
1. Make sure Java8 is in the classpath
2. Run the main Java class /DigioAssignment/src/main/java/com/Assignment/LogReader.java
3. The log file is already preset as part of the project

As per the usecase following details from the log file will be extracted and displayed
● The number of unique IP addresses
● The top 3 most visited URLs
● The top 3 most active IP addresses

4. Package also consists of test cases
